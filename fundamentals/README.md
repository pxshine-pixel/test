# 基本面汇总排序

把 `/score` 等来源「打分后的股票」粘贴进来，自动识别「基本面分数」列并排序、排名、汇总。
纯前端、零依赖，数据只在本地浏览器处理。

## 功能

- **多格式导入**：自动识别并解析
  - JSON 数组：`[{"name":"茅台","code":"600519","score":85,"pe":30}]`
  - JSON 映射：`{"茅台":85,"五粮液":78}`
  - Markdown 表格 / CSV / TSV（带表头）
  - 每行「名称 分数」：`贵州茅台 85`
- **自动识别列**：分数（score/评分/基本面/得分…）、名称、代码
- **按分排序排名**：默认按基本面分降序，点任意表头切换排序列与升降序
- **分数色阶**：分数列按高→低渲染绿→红淡色背景
- **汇总统计**：股票数、已评分数、平均分、最高/最低
- **导出 CSV**、**本地保存**（刷新不丢）

## 关于 /score

本工具不产生分数，只做汇总与排序。基本面分数来自你自己的 `/score`（你本地 Claude
环境里的 skill）。把它的输出粘贴进来即可——若分数列没被自动识别，用右上角「排序列」
手动指定。

> 若 `/score` 的输出格式比较特殊导致解析不理想，把一段示例发我，可针对性增强解析。

## 文件结构

- `index.html` / `styles.css` — 界面
- `fundamentals.js` — 解析与排序核心（浏览器 / Node 通用）
- `app.js` — 界面逻辑、色阶、CSV 导出、本地存储
- `fundamentals.test.js` — 单元测试（`node fundamentals.test.js`）

## 测试

```bash
node fundamentals.test.js     # 11 项：各格式解析、数值转换、排序、统计
```

> 数据仅在本地浏览器处理与保存，不上传。基本面分数由 /score 提供，本工具不构成投资建议。
