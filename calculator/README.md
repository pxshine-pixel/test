# 股价计算器

一个零依赖的网页版股票计算器，用于计算 A 股交易的成本、盈亏、保本价与目标卖出价（含手续费）。

## 功能

- **买入成本**：买入金额、佣金（含最低佣金）、过户费、总成本、每股成本
- **盈亏分析**：按卖出价计算卖出净收入、盈亏金额与收益率（卖出含印花税）
- **保本价**：考虑全部费用后、卖出不亏损所需的最低价格（精确处理最低佣金）
- **目标价**：按设定的目标收益率反推需要的卖出价格

## 使用

直接用浏览器打开 `index.html` 即可，无需任何构建或安装。

## 品种费率预设

页面提供「品种预设」下拉，选中后自动按下表填入佣金 / 最低佣金 / 印花税 / 过户费，
也可手动修改（改动后自动切换为「自定义」）：

| 品种 | 佣金 | 最低佣金 | 印花税(卖出) | 过户费(双向) | 备注 |
| --- | --- | --- | --- | --- | --- |
| 沪深股票 | 万1（50万以上万0.854） | 不免5（0） | 0.05% | 0.001% | 勾选大资金档切换万0.854 |
| 场内基金 ETF/REITs | 万0.5 | 0 | 无 | 无 | 资金量大可议至更低 |
| LOF 基金 | 万1（不打折） | 0 | 无 | 无 | |
| 债基 / 货币基金 | 0 | 0 | 无 | 无 | 交易与申赎 0 手续费 |
| 可转债·上交所 | 万0.5 | 0 起收 | 无 | 无 | |
| 可转债·深交所 | 万0.5 | 0.1 元起收 | 无 | 无 | |
| 港股通 | 万1 | 无最低 | — | — | 港股印花税/交易费等未计入，仅供参考 |
| 北交所 | 万3（500万以上万2） | 0 | 0.05% | 0.001% | 勾选大资金档切换万2 |

> 默认品种为「沪深股票」（万1、不免5）。佣金、印花税、过户费均以百分比输入，
> 例如「万1」= 0.01%。各档费率以券商实际收费为准。

## 测试

核心计算逻辑在 `calculator.js`，纯函数实现，可在 Node 下直接测试：

```bash
node calculator.test.js
```

## 文件结构

- `index.html` — 页面结构
- `styles.css` — 样式
- `calculator.js` — 计算核心（浏览器 + Node 通用）
- `app.js` — 页面交互与渲染
- `calculator.test.js` — 单元测试

> 本工具不构成投资建议，计算结果以券商实际收费为准。
