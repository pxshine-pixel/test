# 技术分析

基于历史 K 线计算**技术形态**、**密集成交区**与**买卖阶段信号**。行情通过本机富途 OpenD 桥接获取（与个股长期跟踪系统共用 `bridge.py`），并内置示例数据，未连接也能体验。

## 功能

### 技术形态
- **均线**：MA5 / MA20 / MA60，判断多头 / 空头排列
- **MACD**：DIF / DEA / 柱状图，金叉死叉
- **RSI(14)**：超买（≥70）/ 超卖（≤30）
- **KDJ**：K / D / J，高低位金叉死叉

### 密集成交区（成交量分布）
把价格区间分桶，按每根 K 线的高低区间把成交量均摊到价位上，得到成交量分布：
- **POC**：成交量最大的价位（筹码峰），常作支撑 / 压力参考
- **价值区间**：约 70% 成交量集中的价带

### 买卖阶段信号
把上述指标规则化打分（每项 −1 / 0 / +1），汇总为综合评分并映射到阶段：

| 评分 | 阶段 |
| --- | --- |
| +3 及以上 | 强势买入区 |
| +1 ~ +2 | 逢低布局 |
| 0 | 震荡观望 |
| −1 ~ −2 | 逢高减仓 |
| −3 及以下 | 卖出区 |

界面会**逐项列出每个信号的方向与理由**，透明可解释，而非黑箱结论。

## 使用

1. 启动 OpenD 与桥接服务（见 [`../tracker/README.md`](../tracker/README.md)）：
   ```bash
   pip install futu-api
   python ../tracker/bridge.py
   ```
2. 浏览器打开 `index.html`，输入代码点「分析」。
3. 想先体验？直接点「**载入示例**」，用内置 K 线查看完整效果。

## 文件结构

- `index.html` / `styles.css` — 界面
- `ta.js` — 技术指标核心（浏览器 / Node 通用）
- `sample.js` — 内置示例 K 线
- `quotes.js` — 桥接客户端（取含成交量的 K 线）
- `app.js` — 界面逻辑与 SVG 图表
- `ta.test.js` — 单元测试（`node ta.test.js`）

## 测试

```bash
node ta.test.js     # 13 项：均线 / MACD / RSI / KDJ / 密集区 / 阶段信号
```

> 所有指标均为机械计算的技术参考，不构成投资建议；行情以 OpenD 实际返回为准。
