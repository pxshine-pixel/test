# 基本面打分

按 **成长性 / 盈利能力 / 财务健康 / 估值** 四维加权，给股票打出 0–100 的基本面分与评级。
混合式：用数据源的财报指标**自动算分**，权重可调，也可**手动覆盖**总分。纯前端、零依赖。

## 打分维度（默认权重）

| 维度 | 看什么 | 权重 |
| --- | --- | --- |
| 📈 成长性 | 营收增速、净利增速 | 25 |
| 💰 盈利能力 | ROE、毛利率、净利率 | 25 |
| 💧 现金流质量 | 经营现金流 / 归母净利（利润含金量） | 20 |
| 🏦 财务健康 | 资产负债率（越低越好） | 15 |
| 🏷️ 估值 | PE、**PEG**、PB（越低越高分，亏损记 0） | 15 |

- 每维 0–100，按权重汇总成总分；**缺失维度自动剔除并重新归一化**。
- 评级：≥80 优 / ≥65 良 / ≥50 中 / ≥35 偏弱 / 其余 弱。
- **估值含 PEG**：高增长容忍高 PE，避免高成长股被高 PE 误杀（PEG = PE / 净利增速）。

### 权重预设

顶部「预设」可一键切换风格：**均衡 / 价值 / 成长 / 质量**；也可手动改每维权重（如只看成长把估值调 0）。某只想人工定调时，填「手动」列覆盖自动分。

### 红旗警示

自动标出风险信号（鼠标悬停看说明）：**亏损、增收不增利、高杠杆（负债>70%）、现金流存疑（现金流/净利<30%）、利润大幅下滑（净利同比<-50%）**。让分数不是黑箱。

## 用法

1. 「添加」录入代码（名称可选）。
2. 指标可**手动填**，也可点行尾「⤓」从数据源**自动拉取**（PE/PB 来自行情，营收/净利增速/ROE/毛利率/净利率/负债率来自财报）。「⤓ 拉取全部」批量拉。
3. 表格按最终分**自动排序**，分数列按高→低绿→红着色。
4. 「导出 CSV」保存打分结果（可再喂给「基本面汇总排序」）。

> 数据源为本机 `bridge.py` 或云端 `data-worker`；**Shift/Alt + 点击**右上角状态可设置地址。

## ⚠️ 说明

阈值是 A 股经验值，**不同行业差异大**（银行/地产高负债属正常、周期股低 PE 未必便宜、重整/题材股纯指标会失真）。结果仅供**横向初筛**参考，不构成投资建议。数据仅存本地浏览器。
