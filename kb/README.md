# 基本面知识库

像知识库一样按股票存档基本面分析：**左边股票名称 + 代码列表，右边基本面分析报告**。
纯前端、零依赖，数据只存本地浏览器。

## 功能

- **两栏式布局**：左侧自选股列表（名称 + 代码、可搜索），右侧该股的基本面报告
- **Markdown 报告**：右侧报告用 Markdown 编写并渲染（标题/表格/列表/引用/粗斜体/链接）
- **来自 /score**：把 `/score` 的分析直接粘贴进报告即可
- **自动拉取指标**：点「拉取指标」从本机 OpenD 取名称与 PE/PB/股息等，生成报告骨架；
  新增股票时若桥接在线会自动尝试拉取
- **搜索 / 本地保存 / 导入导出**：localStorage 持久化，可导出 JSON 备份

## 使用

直接用浏览器打开 `index.html`：
1. 左上角输入代码（名称可留空，自动从 OpenD 取）→ 添加
2. 右侧点「编辑」写报告，或粘贴 `/score` 的输出；点「拉取指标」生成骨架
3. 行情功能需本机 `bridge.py`（见 [`../tracker/README.md`](../tracker/README.md)）；
   不连桥接也能纯手动记录

## 与「基本面汇总排序」的区别

- 本工具（`kb/`）：**按股票存档报告**，知识库式查阅。
- `fundamentals/`：把 `/score` 的**打分结果排序排名**。
两者互补，可配合使用。

## 文件结构

- `index.html` / `styles.css` — 两栏界面
- `kb.js` — 核心：搜索、报告模板、Markdown 渲染（浏览器 / Node 通用）
- `quotes.js` — OpenD 桥接客户端
- `app.js` — 界面逻辑、本地存储
- `kb.test.js` — 单元测试（`node kb.test.js`）
