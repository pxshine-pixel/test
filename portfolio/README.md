# 投资组合追踪器

一个零依赖的网页应用，用于管理多只持仓、自动计算总盈亏，并用饼图展示资产配置。

## 功能

- **多只持仓管理**：表格内直接编辑名称、数量、成本价、现价
- **自动汇总**：总市值、总成本、总盈亏、总收益率，盈亏用红绿色区分
- **资产配置饼图**：纯 SVG 绘制，按市值占比展示各持仓权重
- **本地保存**：数据存于浏览器 `localStorage`，刷新不丢、不上传
- **一键示例 / 清空**：快速体验或重新开始

## 使用

用浏览器打开 `index.html` 即可，无需安装或构建。现价需手动更新。

## 文件结构

- `index.html` — 页面结构
- `styles.css` — 样式
- `portfolio.js` — 计算核心（浏览器 / Node 通用）
- `app.js` — 表格编辑、本地存储与 SVG 饼图渲染
- `portfolio.test.js` — 单元测试

## 测试

```bash
node portfolio.test.js
```

> 数据只保存在本地浏览器，不会上传。本工具不构成投资建议。
